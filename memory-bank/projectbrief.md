# Jetimworks - Project Brief

## Overview
Jetimworks is a personal/professional consulting website showcasing technical expertise and services. The site features a unique terminal-inspired interface that feels like interacting with a command-line system.

## Core Requirements
- Terminal-style UI with command input navigation
- 3D page flip transitions between sections
- Dark theme with green-on-black Matrix aesthetic
- Mobile-responsive design
- Multiple page sections: Hero, ShowRoom, ContactUs, FunFact, Products

## Target Users
- Potential clients seeking consulting services
- Tech enthusiasts appreciating the terminal aesthetic
- Anyone wanting to learn about the services offered

## Design Language
- **Colors**: Black background (#0a0f0a), Green accent (#00ff41)
- **Fonts**: Bruno Ace SC (headers), Press Start 2P (terminal text), Space Mono (body)
- **Effects**: Binary rain background, typing animation, dizzy text effect, 3D page flips

## Tech Stack
- React 18.2.0
- Vite 5.0.0
- Framer Motion 10.16.4 for animations
- No CSS framework - custom CSS styling

## Features
1. **Hero Section**: Terminal with typing text, command list, autocomplete input
2. **ShowRoom**: Project showcase with image and description
3. **ContactUs**: Email link and Calendly booking integration
4. **FunFact**: Random tech facts with refresh functionality
5. **Products**: Placeholder page for future products

## Pages/Sections
- `/show-room` - Project showcase
- `/contact-us` - Contact information
- `/fun-fact` - Random tech facts
- `/products` - Coming soon page

## Success Criteria
- Smooth 3D flip animations between pages
- Functional command input with autocomplete
- Responsive design for mobile devices
- Fast loading and smooth performance
